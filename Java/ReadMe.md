## Instalação Java, JDK e Maven

Primeiro, instala o jdk 
Coloca nas variáveis do sistema uma variável chamada JAVA_HOME com o caminho do jdk, que comumente fica numa pasta Java do program files e edita o Path com o caminho do jdk + /bin 

Depois disso, instala o mvn tmbm no site deles e coloca o path também
