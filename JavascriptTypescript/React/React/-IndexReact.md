# React

React é uma linguagem front end declarativa, resumindo, é fácil você dizer para ela o que você vai fazer, não precisa especificar tudo. Assim como não precisamos usar react em toda aplicação, podemos definir onde usar react e podemos usar o React Native para aplicativos mobile

Para fazer build = `npm run build`
Para rodar o projeto = `npm run start`

* React não precisa estar na página web inteira.
* ReactDom = é quem é responsável por manipular o DOM.
